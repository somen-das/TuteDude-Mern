Color Units in CSS

Color Units

In CSS, colors can be specified using various units and formats. Here are some common color units:

  • Hexadecimal (Hex): A six-digit code representing the red, green, and blue components of a color. For example, #FF5733.
  • RGB: Stands for Red, Green, Blue. It uses the format rgb(255, 87, 51) where each value ranges from 0 to 255.
  • RGBA: Similar to RGB but includes an alpha channel for opacity. The format is rgba(255, 87, 51, 0.5) where the last value ranges from 0 (fully transparent) to 1 (fully opaque).
  • HSL: Stands for Hue, Saturation, Lightness. The format is hsl(9, 100%, 60%) where hue is an angle (0-360), saturation and lightness are percentages.
  • HSLA: Similar to HSL but includes an alpha channel for opacity. The format is hsla(9, 100%, 60%, 0.5).
  • Named Colors: CSS also supports named colors like "red", "blue", "green", etc.
